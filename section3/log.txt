Prompt: Write a Python function to compute factorial.

Answer:

Certainly! Here's a simple Python function to calculate the factorial of a given number using recursion:

```python
def factorial(n):
    """
    Calculate the factorial of a given number n.

    Args:
        n (int): A non-negative integer whose factorial is to be calculated.

    Returns:
        int: The factorial of n.
    """

    # Base case: if n is 0 or 1, return 1
    if n == 0 or n == 1:
        return 1

    # Recursive case: multiply n with the factorial of (n-1)
    else:
        return n * factorial(n - 1)

# Example usage and testing
if __name__ == "__main__":
    num = 5
    print(f"The factorial of {num} is {factorial(num)}")
```

### Explanation:
- **Base Case**: If `n` is either 0 or 1, the factorial is defined as 1.

def factorial(n):
    """
    Compute the factorial of a given number n.

    Parameters:
    n (int): The number to compute the factorial of.

    Returns:
    int: The factorial of n.
    """
    if n < 0:
        raise ValueError("Factorial is not defined for negative numbers.")
    elif n == 0:
        return 1
    else:
        result = 1
        for i in range(1, n + 1):
            result *= i
        return result
